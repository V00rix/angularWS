Over the past month, our small new generation toys production start-up has become very popular, so we decided to expand. With the growing number of customers, we are in need to provide a convenient way to order goods.

Final goal:
We need a web site - an integral and independent solution that we could host on our server. The website should fulfill the following objectives:
1. The ability for all users to view all information about the products provided by us, ie:
- cost
- availability (if the product is not available - still provide information about it)
- product rating (based on user feedback)
- user reviews of the product
2. Ability to register on the site for the following:
2.1. Possibility of ordering goods
2.2. Opportunity to leave feedback
2.3. Ability to delete your account

Environment:
The server part must be executed in php. Supported version: <= 7.0
The web site should present itself a Single-Page-Application, be responsive, correctly displayed on all browsers.
The design should be intuitive and user-friendly.
